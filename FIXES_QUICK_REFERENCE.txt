# QUICK REFERENCE - Fixes Applied

## TWO CRITICAL FIXES COMPLETED

### 🔴 Fix #1: False Positive Violations (WINDOW MINIMIZE)

**Problem:** Minimize/maximize window counted as cheating
**Solution:** Only record violations hidden for >2 seconds (not <1s minimize)
**File Modified:** `exam-monitor-ENHANCED.js`
**Status:** ✅ DONE

**How it works:**
- Minimize window = document hidden for <1 second = IGNORED ✓
- Switch tab = document hidden for >2 seconds = VIOLATION ✓

---

### 🔴 Fix #2: Results Not Recording (DURATION & TIMES)

**Problem:** Admin results page didn't show exam duration or times
**Solution:** Calculate, store, and display exam duration from start to end
**Files Modified:** `app.py`, `admin_results.html`
**Status:** ✅ DONE

**What's now recorded:**
- ✅ exam_start_time
- ✅ exam_end_time
- ✅ exam_duration_seconds
