🎯 TECHNICAL SUMMARY:

ROOT CAUSE (Masalah Awal):
  Exam dalam NEW window (separate JS context)
  → Event listeners dalam parent window
  → Tidak bisa detect events di exam window
  → Violations = 0

SOLUTION (Solusi):
  Exam wrapper dalam SAME window dengan monitor
  → Event listeners dalam exam window context
  → Semua events TERDETEKSI
  → Violations RECORDED

ARCHITECTURE:
  
  OLD (BROKEN):
  Dashboard → window.open() → Exam (separate context) ❌
  
  NEW (FIXED):
  Wrapper (proctored-exam.html) 
    ├─ Monitor Bar (top)
    ├─ exam-monitor-ENHANCED.js (ACTIVE)
    └─ Exam Content (iframe)
       → All events detected ✅
